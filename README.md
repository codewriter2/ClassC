## 📖 단원별 개요

### 1. 변수와 자료형
> 프로그래밍의 기초가 되는 데이터 저장과 표현 방법을 학습한다. 변수의 개념, 다양한 자료형(정수형, 실수형, 문자형), 상수, 기본 입출력 함수를 다룬다.

### 2. 수식과 연산자
> C언어에서 제공하는 다양한 연산자를 학습한다. 산술, 대입, 관계, 논리, 비트, 조건 연산자와 연산자 우선순위, 형변환을 이해한다.

### 3. 조건문
> 프로그램의 흐름을 조건에 따라 분기하는 방법을 학습한다. if문, if-else문, 다중 조건문(if-else if), switch-case문을 다룬다.

### 4. 반복문
> 특정 코드를 반복 실행하는 구조를 학습한다. for문, while문, do-while문의 차이와 활용, break와 continue를 이해한다.

### 5. 배열
> 동일한 자료형의 데이터를 연속적으로 저장하는 배열을 학습한다. 1차원 배열과 2차원 배열의 선언, 초기화, 활용을 다룬다.

### 6. 함수
> 코드를 모듈화하여 재사용하는 함수를 학습한다. 함수의 정의, 호출, 매개변수 전달, 반환값, 변수의 범위, 재귀 함수를 다룬다.

### 7. 포인터
> C언어의 핵심 개념인 포인터를 학습한다. 메모리 주소의 개념, 포인터 변수, 간접 참조, 포인터와 배열/함수의 관계를 이해한다.

### 8. 문자열
> 문자의 배열인 문자열을 학습한다. 문자열의 저장 방식, 입출력, 표준 라이브러리 문자열 처리 함수들을 다룬다.

### 9. 구조체
> 서로 다른 자료형을 하나로 묶는 구조체를 학습한다. 구조체의 정의, 변수 선언, 배열, 포인터, typedef를 다룬다.

### 10. 파일 입출력
> 파일을 통해 데이터를 읽고 쓰는 방법을 학습한다. 텍스트 파일과 이진 파일의 처리, 파일 관련 함수들을 다룬다.

### 11. 동적 메모리
> 프로그램 실행 중 메모리를 할당하고 해제하는 방법을 학습한다. malloc, calloc, realloc, free 함수와 메모리 관리를 다룬다.

### 12. 전처리와 분할 컴파일
> 컴파일 전에 처리되는 전처리기 지시문과 대규모 프로그램의 분할 컴파일 방법을 학습한다. 매크로, 조건부 컴파일, 헤더 파일을 다룬다.
