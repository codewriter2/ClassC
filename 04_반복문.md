## 1. 반복문이란?

### 반복의 필요성

반복은 어떤 단계를 반복하게 하는 것으로, 반복 구조를 사용하면 프로그램이 **간단하고 빠르게** 된다. 동일한 작업을 반복하기 위하여 똑같은 문장을 복사하여 붙여넣기 하는 것보다 **반복 구조를 사용하는 편이 프로그램을 간결하게** 만든다.

### 반복문 없이 코딩하면?

```c
#include <stdio.h>
int main(void)
{
    int meter;
    meter = 0 * 1609;
    printf("0 마일은 %d미터입니다\n", meter);
    meter = 1 * 1609;
    printf("1 마일은 %d미터입니다\n", meter);
    meter = 2 * 1609;
    printf("2 마일은 %d미터입니다\n", meter);
    // ... 같은 처리 과정이 계속 반복됨!
    return 0;
}
```

> ⚠️ 이렇게 코드를 반복하면 비효율적이고 유지보수가 어렵다!

### 반복문의 종류

| 반복문 | 특징 |
|:---:|------|
| `while` | **조건 제어 반복** — 특정한 조건이 만족되면 계속 반복 |
| `do-while` | **조건 제어 반복** — 최소 1번은 실행 후 조건 검사 |
| `for` | **횟수 제어 반복** — 정해진 횟수만큼 반복 |

## 2. while 문

### 기본 구조

```c
while (조건식)
{
    반복할 문장들;
}
```

- `while`문은 **조건식이 참이면** 문장들을 반복적으로 실행한다.
- 조건식이 **거짓이 되면** 반복을 종료하고 다음 문장으로 넘어간다.

### 예제: 10번 반복 출력

```c
#include <stdio.h>
int main(void)
{
    int i = 0;
    while (i < 10)
    {
        printf("수업 시간에 떠들지 않겠습니다.\n");
        i++;
    }
    return 0;
}
```

### 예제: 반복 제어 변수 출력

```c
#include <stdio.h>
int main(void)
{
    int i = 0;
    while (i < 10)
    {
        printf("i=%d  ", i);
        i++;
    }
    return 0;
}
```

**실행 결과:**
```
i=0  i=1  i=2  i=3  i=4  i=5  i=6  i=7  i=8  i=9
```

### 예제: 마일을 미터로 변환 (while 버전)

```c
#include <stdio.h>
int main(void)
{
    int meter;
    int i = 0;
    while (i < 3)
    {
        meter = i * 1609;
        printf("%d 마일은 %d미터입니다\n", i, meter);
        i++;
    }
    return 0;
}
```

### 예제: 팩토리얼 구하기

**Factorial**: 1부터 어떤 양의 정수 n까지의 정수를 모두 곱한 것 (n!로 표기)
- 0! = 1
- 3! = 3 × 2 × 1 = 6
- n! = n × (n-1) × (n-2) × ... × 2 × 1

```c
int i = 5;
int factorial = 1;
while (i >= 1)          // i를 감소시키면서 i가 1이상이면 반복
{
    factorial *= i;
    i--;
}
// factorial = 120
```

### 예제: 구구단 출력 (while 버전)

```c
// while 문을 이용한 구구단 출력 프로그램
#include <stdio.h>
int main(void)
{
    int n;
    int i = 1;
    printf("구구단 중에서 출력하고 싶은 단을 입력하시오: ");
    scanf("%d", &n);
    while (i <= 9)
    {
        printf("%d*%d = %d \n", n, i, n * i);
        i++;
    }
    return 0;
}
```

**실행 결과 (9 입력 시):**
```
구구단 중에서 출력하고 싶은 단을 입력하시오: 9
9*1 = 9
9*2 = 18
9*3 = 27
...
9*9 = 81
```

### 예제: 1부터 1000까지의 합

```c
#include <stdio.h>
int main(void)
{
    int i, sum;
    i = 1;
    sum = 0;
```

## 3. do-while 문

### 기본 구조

```c
do
{
    반복할 문장들;
} while (조건식);    // ← 세미콜론(;) 필수!
```

- `while`문과 비슷하나 반복 조건을 **루프의 끝에서 검사**한다는 것이 다르다.
- **적어도 한 번은** 반복 문장을 실행한다.

### while vs do-while 비교

| 구분 | `while` | `do-while` |
|:---:|---------|------------|
| 조건 검사 시점 | **루프 시작** 전 | **루프 종료** 후 |
| 최소 실행 횟수 | 0번 (조건이 거짓이면 미실행) | **1번** (무조건 한 번 실행) |
| 세미콜론 | 불필요 | `while(조건);` 뒤에 **`;` 필수** |

### 예제: 메뉴 입력 받기

`do-while`문은 **입력을 처리하는 부분**에서 많이 사용된다.

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
int main(void)
{
    int i = 0;
    do
    {
        printf("1---파일열기\n");
        printf("2---파일저장하기\n");
        printf("3---종료\n");
        printf("하나를 선택하시오: ");
        scanf("%d", &i);
    } while (i < 1 || i > 3);    // 유효한 값이 아닌 동안 반복
    printf("선택된 메뉴=%d\n", i);
    return 0;
}
```

### Lab: 숫자 추측 게임

프로그램이 가지고 있는 정수를 사용자가 알아맞히는 게임. 사용자가 답을 제시하면 프로그램은 자신이 저장한 정수와 비교하여 제시된 정수가 **더 높은지 낮은지**만을 알려준다.

**알고리즘:**
```
do
    사용자로부터 숫자를 guess로 입력받는다.
    시도횟수를 증가한다.
    if (guess < answer)
        숫자가 낮다고 출력한다.
    if (guess > answer)
        숫자가 높다고 출력한다.
while (guess != answer);
"축하합니다"와 시도횟수를 출력한다.
```

## 4. for 문

### 기본 구조

```c
for (초기식; 조건식; 증감식)
{
    반복할 문장들;
}
```

- **초기식**: 반복 시작 전 한 번만 실행 (변수 초기화)
- **조건식**: 매 반복 전에 검사 (참이면 계속, 거짓이면 종료)
- **증감식**: 매 반복 후에 실행 (변수 값 변경)
- **정해진 횟수만큼** 반복하는 구조에 적합

### 예제: 1부터 10까지의 합

```c
int i, sum;
sum = 0;
for (i = 1; i <= 10; i++)       // i는 1부터 10까지 증가
    sum += i;                    // sum = sum + i;와 같음
```

### 예제: 사각형 그리기

```c
// 반복을 이용한 네모 그리기
#include <stdio.h>
int main(void)
{
    int i;
    printf("**********\n");
    for (i = 0; i < 5; i++)
        printf("*        *\n");
    printf("**********\n");
    return 0;
}
```

### 예제: 팩토리얼 구하기 (for 버전)

```c
#include <stdio.h>
int main(void)
{
    int fact = 1;
    int i, n;
    printf("정수를 입력하시요: ");
    scanf("%d", &n);
    for (i = 1; i <= n; i++)
        fact = fact * i;
    printf("%d!은 %d입니다.\n", n, fact);
    return 0;
}
```

**실행 결과 (5 입력 시):**
```
정수를 입력하시요: 5
5!은 120입니다.
```

### for 문에서 주의할 점

1. **조건식에 논리 연산자 사용 가능**
   ```c
   for (i = 0; i < 100 && sum < 2000; i++)
       sum += i;
   ```

2. **3부분이 전부 비어 있으면 무한 루프**
   ```c
   for ( ; ; )
       printf("Hello World!\n");   // 무한 반복
   ```

3. **감소 수식도 사용 가능**
   ```c
   for (i = 10; i > 0; i--)
       printf("Hello World!\n");
   ```

4. **반복 문장이 2줄 이상이면 중괄호 `{}` 필수**

## 5. 중첩 반복문

### 중첩 반복문이란?

**중첩 반복문(nested loop)**: 반복문 안에 다른 반복문이 위치하는 구조

- 바깥 반복문이 1번 실행될 때, 안쪽 반복문은 처음부터 끝까지 모두 실행된다.

### 예제: 사각형 그리기 (중첩 for)

```c
// 중첩 for 문을 이용하여 *기호를 사각형 모양으로 출력
#include <stdio.h>
int main(void)
{
    int x, y;
    for (y = 0; y < 5; y++)        // 바깥 반복문 (행)
    {
        for (x = 0; x < 10; x++)   // 안쪽 반복문 (열)
            printf("*");
        printf("\n");              // 안쪽 반복문이 종료될 때마다 줄바꿈
    }
    return 0;
}
```

**실행 결과:**
```
**********
**********
**********
**********
**********
```

### Lab: 구구단 전체 출력 (중첩 for)

```c
#include <stdio.h>
int main(void)
{
    int i, k;
    for (i = 1; i <= 9; i++) {
        for (k = 1; k <= 9; k++) {
            printf("%d X %d = %d \n", i, k, i * k);
        }
    }
    return 0;
}
```

## 6. break 문

### break 문이란?

`break`문은 반복 루프를 **즉시 빠져나오는 데** 사용된다.

```c
for (...) {
    ...
    if (조건)
        break;     // 반복문 즉시 종료
    ...
}
// break 이후 여기로 이동
```

### 예제: 음수 입력 시 조기 종료

사용자로부터 10개의 실수를 받아서 합계를 출력한다. 사용자가 중간에 **음수를 입력하면** 반복 루프를 조기에 종료시킨다.

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
int main(void)
{
    int i;
    double number, sum = 0.0;
    for (i = 1; i <= 10; i++)
    {
        printf("%d번째 실수를 입력하시오: ", i);
        scanf("%lf", &number);
        // 사용자가 음수를 입력하면 반복 루프가 종료된다.
        if (number < 0.0)
            break;
        sum += number;
    }
    printf("합계 = %f", sum);
    return 0;
}
```

## 7. continue 문

### continue 문이란?

`continue`문은 **현재의 반복을 중단**하고 **다음 반복을 시작**하게 한다.

```c
for (...) {
    ...
    if (조건)
        continue;   // 아래 코드를 건너뛰고 다음 반복으로
    ...             // ← continue 시 이 부분이 건너뛰어진다
}
```

### break vs continue 비교

| 구분 | `break` | `continue` |
|:---:|---------|------------|
| 동작 | 반복문 **전체 종료** | **현재 반복만** 건너뜀 |
| 이후 실행 | 반복문 다음 문장 실행 | 다음 반복(증감식) 실행 |

### 예제: 짝수만 출력

```c
#include <stdio.h>
int main(void)
{
    int i;
    for (i = 0; i < 10; i++) {
        if (i % 2 == 1)         // 홀수이면
            continue;           // 다음 반복으로 건너뜀
        printf("정수: %d \n", i);
    }
    return 0;
}
```

**실행 결과:**
```
정수: 0
정수: 2
정수: 4
정수: 6
정수: 8
```

### Lab: 산수 문제 자동 출제

초등학교 학생들을 위한 산수 문제를 자동으로 출제하는 프로그램을 작성. 덧셈 문제들을 자동으로 생성하며, 피연산자는 0~99 사이의 난수. **한 번이라도 맞으면 종료**한다.

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <stdlib.h>

int main(void)
{
    int i, ans;
    printf("산수 문제를 자동으로 출제합니다.\n");
    while (1) {
        int x = rand() % 100;
        int y = rand() % 100;
        printf("%d + %d = ", x, y);
        scanf("%d", &ans);
        if (x + y == ans) {
            printf("맞았습니다.\n");
            break;
        }
        else
            printf("틀렸습니다.\n");
    }
    return 0;
}
```

### Lab: 소수 찾기

반복 구조를 사용하여 1에서 n 사이의 모든 **소수(prime number)** 를 출력하는 프로그램.

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main()
{
    int n, is_prime;
    printf("어디까지 찾을까요? : ");
    scanf("%d", &n);

    for (int i = 2; i <= n; i++) {
        is_prime = 1;
        for (int k = 2; k < i; k++) {
            if (i % k == 0) {
                is_prime = 0;
                break;              // 약수를 발견하면 더 이상 검사 불필요
            }
        }
        if (is_prime == 1)
            printf("%d, ", i);
    }
    printf("\n\n");
    return 0;
}
```

**실행 결과 (20 입력 시):**
```
어디까지 찾을까요? : 20
2, 3, 5, 7, 11, 13, 17, 19,
```

## 핵심 정리

| 주제 | 핵심 내용 |
|------|--------|
| **반복문의 종류** | `while`, `do-while`, `for` |
| **while** | 조건이 참인 동안 반복. 0번 이상 실행 가능 |
| **do-while** | 최소 1번 실행 후 조건 검사. 입력 처리에 적합 |
| **for** | 초기식·조건식·증감식을 한 줄에. 횟수 제어 반복에 적합 |
| **무한 루프** | `while(1)` 또는 `for(;;)` |
| **중첩 반복문** | 반복문 안에 반복문 (구구단, 사각형 그리기 등) |
| **break** | 반복문 **전체 종료** |
| **continue** | **현재 반복만** 건너뛰고 다음 반복 진행 |
